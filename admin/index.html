✦ Como funciona o painel
Edite qualquer texto, imagem ou link nos menus à esquerda. Quando terminar, clique em Salvar e publicar — as alterações serão enviadas ao GitHub e o Vercel fará o deploy automaticamente em cerca de 30 segundos.
Primeiro acesso — configuração inicial
Criar repositório no GitHub
Crie um repositório privado em github.com/new e faça upload de todos os arquivos do site para ele.
Gerar Token do GitHub
Em Settings → Developer Settings → Tokens (classic), crie um token com permissão repo. Guarde — só aparece uma vez.
Conectar ao Vercel
Em vercel.com, importe o repositório. O Vercel detectará os arquivos e fará o primeiro deploy.
Configurar variáveis de ambiente no Vercel
No painel do Vercel → Settings → Environment Variables, adicione:
GITHUB_TOKEN → seu token gerado
GITHUB_OWNER → seu usuário GitHub
GITHUB_REPO → nome do repositório
ADMIN_PASSWORD → senha deste painel
Configurar a URL da API neste painel
Após o deploy, edite o arquivo admin/index.html e substitua SEU-DOMINIO.vercel.app pela URL real do seu projeto Vercel.
