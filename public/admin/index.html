🗺️ Como o site funciona

O seu site é formado por 5 serviços diferentes que trabalham integrados. Veja o fluxo das informações:

🧰 Os 5 Serviços e Suas Funções
1. Domínio

GoDaddy

Onde está registrado o nome do site (aescolhidadadeusa.site). O endereço que as leitoras digitam.

  • Pagar a renovação anual do domínio
  • Não ignore os avisos de vencimento da GoDaddy
Acessar GoDaddy
2. Hospedagem

Vercel

Plataforma que coloca o site no ar. Toda alteração no GitHub é publicada aqui automaticamente em 2 min.

  • Monitorar se o site está online
  • Configurar chaves secretas (env vars)
Acessar Vercel
3. Código

GitHub

Onde fica guardado o código-fonte do site. As alterações de links são salvas aqui em um arquivo json.

  • Cofre de segurança do código
  • Permite reverter alterações erradas
Ver Repositório
4. Banco de Dados

Supabase

Banco de dados que guarda os e-mails e senhas criptografadas das leitoras cadastradas.

  • Ver leitoras cadastradas se o painel falhar
  • Restaurar banco se ele pausar por inatividade
Acessar Supabase
5. E-mails

Resend

Serviço responsável por disparar o e-mail de confirmação assim que a leitora se cadastra no site.

  • Ver se os e-mails estão sendo entregues
  • Monitorar limite diário de e-mails (100/dia)
Acessar Resend
Variável na Vercel Função / O que faz

🚨 Problemas Comuns e Como Resolver

Se algo der errado com o site, verifique estas soluções antes de acionar o suporte técnico:

❌ O site está fora do ar (erro de conexão ou página não encontrada)
Possíveis causas:
  • O domínio (GoDaddy) venceu ou não foi pago.
  • O servidor de hospedagem (Vercel) teve uma falha na publicação de alguma alteração.
O que fazer:
  1. Acesse o GoDaddy.com e veja se o domínio aescolhidadadeusa.site está ativo e pago.
  2. Acesse a Vercel.com, clique no projeto e vá na aba Deployments. Se o último deploy tiver uma bola vermelha (falha), clique para ler os logs do erro.
❌ As leitoras não conseguem se cadastrar na Área de Membros
Possíveis causas:
  • O banco de dados (Supabase) está pausado por falta de uso temporário (projetos gratuitos pausam após 1 semana sem cadastros).
  • A conexão entre o site e o Supabase falhou.
O que fazer:
  1. Acesse o Supabase.com e faça login.
  2. Veja se o seu projeto está ativo. Se estiver com status de "Pausado" (Paused), clique no botão "Restore Project" (Restaurar Projeto). Ele voltará a funcionar em poucos minutos.
❌ O e-mail de confirmação de cadastro não chega para a leitora
Possíveis causas:
  • O e-mail caiu na pasta de Lixo Eletrônico ou Spam da leitora.
  • Você ultrapassou o limite do plano gratuito da ferramenta de envio de e-mails (Resend permite até 100 envios por dia).
O que fazer:
  1. Peça para a leitora verificar o lixo eletrônico ou a aba "Promoções" do e-mail.
  2. Acesse o Resend.com e verifique o gráfico de envios na página inicial para ver se a conta está ativa e se o limite do dia foi atingido.
❌ O painel admin diz "senha incorreta" ou não abre
Possíveis causas:
  • A senha foi digitada com espaços ou letras maiúsculas/minúsculas incorretas.
  • A senha cadastrada nas variáveis de ambiente do site na Vercel foi modificada.
O que fazer:
  1. Faça login na Vercel.com.
  2. Entre no projeto aescolhidadadeusa → aba Settings → Environment Variables.
  3. Clique no botão com formato de "olho" ao lado da variável ADMIN_PASSWORD para ler a senha correta e usá-la para logar.
❌ Salvei um link ou imagem no painel, mas o site não atualizou
Possíveis causas:
  • A publicação das alterações leva cerca de 2 a 3 minutos para finalizar na Vercel.
  • O navegador guardou a versão antiga do site no histórico (cache).
O que fazer:
  1. Aguarde 3 minutos após clicar em salvar.
  2. No computador, recarregue a página segurando Ctrl + Shift + R (no Mac: Cmd + Shift + R) para forçar o navegador a buscar a versão mais recente.
